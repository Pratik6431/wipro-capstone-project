# Playwright Automation - Capstone Project

-> This repository contains the Capstone Project automated testing framework built using **Playwright**. The framework is designed to
   test end-to-end user workflows, functional requirements, and backend APIs across multiple browser contexts.

##  Project Overview
* **Automation Tool:** Playwright (JavaScript)
* **Target Application:** LambdaTest Ecommerce Playground
* **Total Services in Scope:** 8 Core Services
* **Total Test Cases:** 120 Test Cases (15 per service)

## 📁 Project Structure & Documentation
The detailed project proposal, listing all the selected services and their testing scope, has been documented and uploaded to this repository.
* **Proposal Document:** `Capstone_Project_Planner.docx`

## 🛠️ Services Selected for Testing
1. **Authentication & User Management** - Login, registration, and logout flows.
2. **Product Search & Discovery** - Search bars, product grids, sorting, and filters.
3. **Cart Management** - Adding items, quantity updates, subtotals, and checkout initiation.
4. **User Profile Management** - Updating user account details and dashboard validation.
5. **Address Book & Shipping Service** - Managing multiple shipping addresses and zip code validation.
6. **Customer Support & Feedback** - "Contact Us" form submissions and FAQ accordions.
7. **Internal API Testing** - Validating HTTP status codes and JSON payload structures.
8. **System Persistence & State Validation** - Verifying data stability using local storage and cookies after page reloads.
